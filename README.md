# Wanderlust – Travel Listing Platform

Wanderlust is a full-stack travel listing web application inspired by Airbnb where users can explore and manage vacation property listings.

# Live Demo

https://wanderlust-ro2m.onrender.com

# Tech Stack

# Frontend

* HTML
* CSS
* EJS Templates

# Backend

* Node.js
* Express.js

# Database

* MongoDB

# Tools

* Git
* GitHub
* Render

# Features

* User authentication (Login / Signup)
* Create, edit and delete travel listings
* View property listings with images and details
* RESTful routing architecture
* Server-side rendering using EJS

controllers/

# Installation

Clone the repository

git clone https://github.com/Chiranjeevi0106-dev/wanderlust-travel-platform.git

Install dependencies

npm install

Run the application

node index.js

Visit

http://localhost:3000

# Purpose

This project was built to practice full-stack development concepts including REST APIs, MVC architecture, database integration, and authentication.
